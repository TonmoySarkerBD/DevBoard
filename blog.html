Question-1: What are the different ways to select an element in the DOM?


Answer : There are 5 different way to select an element in the DOM (Document Object Model). Those are:
1. getElementById 2. getElementsByClassName 3. getElementsByTagName 4. querySelector 5. querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?


Answer : In HTML, innerHTML, innerText, and textContent are properties of the DOM (Document Object Model). They allow you to read and update the content of HTML elements.
1. innerHTML : The text content of the element, including all spacing and inner HTML tags.
2. innerText : Just the text content of the element and all its children, without CSS hidden text spacing and tags, except script and style elements.
3. textContent : The text content of the element and all descendaces, with spacing and CSS hidden text, but without tags.

Question-3: What is event delegation in the DOM ?


Answer : In the DOM (Document Object Model), "event delegation" is a technique where attach a single event listener to a parent element, allowing it to handle events triggered on its child elements, instead of adding separate listeners to each child element.

Question-4: What is event bubbling in the DOM ?


Answer : Event Bubbling is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.

Question-5: How do you create, add, and remove elements using JavaScript ?


Answer : In JavaScript, we can create, add, and remove elements in the DOM using various ways.
1. Creating an Element : In DOM to create a new element we use document.createElement() method.
2. Adding an Element : In DOM to Add a new element we use appendChild() method .
3. Removing an Element : In DOM to Remove an element we use removeChild() method.
